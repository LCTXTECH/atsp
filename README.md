# Agentic Transaction Security Protocol (ATSP) ## v1.0 · March 2026 · MIT License
ATSP is the open standard for how AI agents declare intent, validate context, and produce immutable decision traces before executing any on-chain transaction.** Published by AgentSentry (agentsentry.net/protocol/atsp). ## Quick Install npm install @agentsentry/atsp ## Core Interface interface ATSPIntentDeclaration { agentId: string; // NHI-scoped principal proposerPubKey:string; // Squads proposer key intentHash: string; // SHA-256 of action sentryVerdict: 'ALLOW' | 'BLOCK' | 'ESCALATE'; decisionTrace: { input: object; reasoning: string; ipiCleared: boolean; }; } ## ATSP Compliance Badge [![ATSP Compliant](https://agentsentry.net/badge/atsp.svg)] (https://agentsentry.net/protocol/atsp) ## Links - Full spec: agentsentry.net/protocol/atsp - elizaOS plugin: npmjs.com/@agentsentry/eliza-plugin - Discord: https://discord.gg/aBX7vbzd
